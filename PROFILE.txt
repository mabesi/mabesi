Cerberus

Solidity, Hardhat, OpenZeppelin, React, Next.js, WebSockets, Node.js, JWT, Jest

A decentralized exchange crypto trader bot project. A Didactic Web3 Automation Platform for DEX Operations.
Cerberus is an educational, multi-tenant SaaS designed to demonstrate practical Web3 automation for decentralized exchange operations. It blends Web2 and Web3 technologies to deliver secure authentication flows, crypto-based billing, hybrid security, and continuous market monitoring, enabling fully automated swap execution on DEX platforms.
The system is structured into four core modules: an engine that tracks market data via WebSocket, a Web2+Web3 backend built with Hardhat and OpenZeppelin, a React/Next.js frontend for user interaction, and a blockchain layer containing the smart contracts that execute DEX operations. The platform includes JWT-based authentication and automated testing with Jest and Hardhat to ensure reliability and robustness.
